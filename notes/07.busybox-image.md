# 06.使用busybox镜像创建容器
这里我们将使用`busybox`镜像创建容器，我们会将其镜像内容提取出来并挂载到启动的容器中。

```Shell
docker pull busybox
docker run -d --name bosybox busybox top -b

docker export -o busybox.tar bosybox
mkdir busybox
tar -xvf busybox.tar -C busybox/
ls -l busybox
```
